English

Back to the minefield is a tribute to and a reinterpretation of Ian Andrew’s Mined-Out game (1983), which was written in Sinclair BASIC for ZX Spectrum.

This new version is being written in Vimclair BASIC for ZX Spectrum 128.

The interface is multilingual: in English, Spanish, Esperanto and Interlingue.

Note: On 2020-12-04 the Git repository was converted to Fossil, and since then GitHub is used only as a copy.
